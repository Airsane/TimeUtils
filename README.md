# Time Utils

Moderni React/Vite aplikace pro prevody casovych jednotek a rychle vypocty s casem. Obsahuje prepinani jazyku (CZ/EN), prevod jednotek a kalkulacku pro pricteni hodin a minut.

## Funkce
- Prevod jednotek (sekundy, minuty, hodiny, dny, tydny, mesice, roky)
- Rozklad dni na roky, mesice a dny pri prevodu dny -> roky
- Pricteni hodin a minut k zadanemu casu (24h format)
- Lokalizace CZ/EN

## Spusteni
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

- React
- TypeScript
- Vite
